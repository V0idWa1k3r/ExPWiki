Humidity System

ExPetrum introduces a system that gives the player an idea of how soon is it going to rain.
Humidity is displayed in the bottom-right corner of the screen with the text representing the percentage of humidity:
  • Very Dry: 0% - 14%
  • Dry: 14% - 28%
  • Normal: 28% - 42%
  • Slightly Wet: 42% - 57%
  • Wet: 57% - 71%
  • Very Wet: 71% - 85%
  • It Will Rain Any Minute: 85% - 100%
  • It's Raining: 100%
Humidity is increased by [0~0.00002] each tick.
Each tick there is a humidity / 10000 chance(0.2% chance each second at 100% humidity) for the rain to start. Thereby it doesn't start raining when humidity reaches 100% but the humidity gets set to 100% when it starts raining.
When the rain starts it will go on for 1000 + [0~20000] ticks.
When the rain stops the humidity is reset to [0~0.33]
