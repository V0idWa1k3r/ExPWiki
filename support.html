Support Beam

Support Beam is a block added by ExPetrum.
Crafting table can be obtained by crafting from 2 Planks and a Saw

Support beams can be placed to prevent gravity-affected blocks like stone or dirt from falling.
A support beam can only be placed on a solid block or another support beam and will drop itself if it's placement is no longer valid.
A single support beam prevents blocks from collapsing in a 7x3x7 cube centered on the support beam.
