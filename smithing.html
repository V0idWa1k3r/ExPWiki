Smithing System


ExPetrum introduces a card-based minigame as it's smithing mechanics.
All minigame data is stored inside the item that is being smithed.
As the item is inserted into the anvil and the recipe is selected a random[2~3] amount of cards will be drawn and shown to the player on the screen. The same card can be drawn multiple times. The player must select a card and then a new set of cards is drawn. The goal is to bring the progress value to the maximum while avoiding the integrity value dropping to 0.

Progress


The green dot represents the progress value that is displayed as current/max. The goal of the minigame is to bring current to max. Most cards increase progress at the cost of integrity or heat.

Inregrity


The blue dot represents the integrity value that is displayed as current/max. If this value drops to 0 the item that is currently being smithed is lost. Most cards will reduce this value although some will give it. If an item is cooled down by right-clicking a trough some[35%~70%] of the lost integrity is restored.

Heat


The red dot represents the heat value of the item. As this value increases the chances to draw Overheat cards also increases linearly. Some cards will increase this value. If an item is cooled down by right-clicking a trough all of the heat is lost(set to 0).

Cards

The following cards exist:

    • -5 Integrity
